# PyGame--Bianca-Habr-dria-Mendes-Marina-Abreu
PyGame - Inspirado no jogo do fogo e da água do friv. Nosso jogo é na mesma linha, porém os personagens serão sol e lua. Além disso, nossos personagens, ao invés de serem parceiros para superar os desafios como no jogo original, serão inimigos e um irá atrapalhar o outro.
